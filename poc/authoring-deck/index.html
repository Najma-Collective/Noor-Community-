Authoring toolkit

Welcome to the authoring preview deck

This proof-of-concept shows how the stage behaves inside a self-contained folder so designers can iterate quickly.

Included in this POC

  • Toolbar controls that mirror production decks.
  • A vertically scrollable canvas to accommodate long authoring notes.
  • Slide navigator integration that is namespaced to this directory.

How to use this space

  1. Duplicate the slide markup below to outline a new story.
  2. Add authoring notes, assets, and layout experiments.
  3. Share the folder directly or open the HTML file in a browser.

Workflow support

Layout checklist

Use this checklist to keep authoring experiments aligned with the production deck system.

Stage structure

  • Keep headlines within two lines when possible.
  • Use the slide-inner wrapper for scrolling.
  • Leverage stacked sections to organise content.

Interactive elements

  • Note any component requirements beside the mockup.
  • List API data needs or external integrations.
  • Flag accessibility considerations early.

Asset tracking

Capture working image URLs inside the slide content. When the deck is promoted, assets can be uploaded to the shared CDN.
