Knowledge check · JSON mapping

Multiple choice quiz

With a partner, choose the best answer for each question, then justify your choice.

  1. Which field ensures the toolbar shows the correct brand label?

    • brand.label
    • title
    • lang

    Correct: brand.label

  2. Where do you declare the layout for each slide?

    • slides[].layout
    • slides[].icon
    • layoutDefaults

    Correct: slides[].layout

  3. How can you provide your media API credential when generating decks?

    • Add pexelsKey to the brief
    • Embed the API key in gallery captions
    • Rename package-lock.json

    Correct: Add pexelsKey to the brief
