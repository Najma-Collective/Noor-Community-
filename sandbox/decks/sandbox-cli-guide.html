Command Flow

  • Invoke

    node scripts/create-deck.mjs --input briefs/your-brief.json --output decks/your-deck.html

  • Watch

    The CLI resolves layout defaults, fetches imagery, and writes polished markup.

Power Tips

  • Reuse

    Store shared snippets in briefs so teams remix slides fast.

  • Preview

    Open the generated HTML locally; rerun the command after each edit for instant updates.

Bonus: add the script to package.json for a memorable npx alias.

Quality Check Before Sharing

Walk through these prompts to ensure your deck dazzles.

Glow

  • Do slides alternate between spacious hero moments and focused content blocks?

  • Will facilitators understand each step without needing extra notes?

Grow

  • Could you swap in activity layouts to make the session more hands-on?

  • Does the deck use approved colors, typography, and partner logos?

Tip: Commit the brief alongside the generated deck for easy iteration.
