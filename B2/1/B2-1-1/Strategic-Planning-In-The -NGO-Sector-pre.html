Model Dialogue: Designing a Digital Skills Program

Read the dialogue between two NGO project managers, Fatima and Yousef. They are discussing a report about local artisans in Nablus.

Fatima: Okay, so the Nablus report is clear. The data confirmed what we had suspected for a while: the core issue is a lack of digital skills, not product quality.

Yousef: Exactly. For the new pilot program, I propose we start with a mentorship model. We could train university students to mentor the artisans one-on-one. For me, the key to long-term sustainability is building those supportive relationships.

Fatima: I see the value in mentorship. However, my only concern is that it might be too slow. The artisans need practical skills now. An alternative approach could be to run intensive, hands-on workshops.

Yousef: That's a fair point. But what's the main advantage of workshops?

Fatima: The evidence strongly suggests that group training builds confidence quickly. We could focus on smartphone photography and social media marketing. My priority would be ensuring a baseline of skills for everyone.

Yousef: I like that. What if we create a hybrid model? We could run the workshops first to teach the core skills, and then the student mentors could provide follow-up support.

Fatima: Now that's a great idea. It's the best of both worlds. Let's start planning the details.
